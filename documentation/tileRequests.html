Tile Paths

Mapsense map tiles are queried according to the commonly used tile coordinate system (with a top-left origin) and are served in the topojson format.

When adding a layer to the map, you need to construct a url path template.

Tile templates take the following format: 

https://{S}-api.mapsense.co/universes/{DATASET}/{Z}/{X}/{Y}.topojson?api-key={KEY}

Where {S} is a particular server (a,b,c, or d -- set by the url function), {TILESET} is the name of a Mapsense tileset, and {KEY} is the api key provided by Mapsense. Use this path as an argument to the url function when defining a layer.

For example: 

var earthLayer = mapsense.topojson().url("https://{S}-api.mapsense.co/universes/mapsense.earth/{Z}/{X}/{Y}.topojson?api-key=key-fb3a8e7a8537477cbe1fb268d7b98c77")

While Mapsense.js expects a url template, you can always access an individual tile by populating its tile coordinates. For example, to get the outermost earth tile: https://a-api.mapsense.co/universes/mapsense.earth/0/0/0.topojson?api-key=key-fb3a8e7a8537477cbe1fb268d7b98c77.
