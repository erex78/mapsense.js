basemap

Create and modify the default Mapsense basemap.
basemap
apiKey
style
mapsense.basemap()
Constructs a new basemap layer, which defaults to loading Mapsense earth tiles.
basemap.apiKey(key)
Sets the key to be used when fetching basemap tiles. The argument is the the API key, which is the alphanumeric string provided by Mapsense. Something like this: "key-2d5eacd8b924489c8ed5e8418bd883bc"
basemap.style(style)
Sets the style to be used for the basemap. If a style name is passed as an argument, all features classes will be prepended with "mapsense-style". If a false value is passed as an argument, feature class names will not be prepended with anything and you can provide your own custom CSS. The preset styles available for use in basemap are:
